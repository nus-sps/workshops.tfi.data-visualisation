2 Some Basic Python

2.1 Python is Case Sensitive 

print('Hello World')
#> Hello World

2.2 x is a variable. ‘x’ is an English letter. 

print('Hello World')
#> Hello World

2.3 We can use f'letters{x}' to combine English with variables. 

print('Hello World')
#> Hello World

2.4 Be mindful of spaces(indentation) at the start of a line. 

print('Hello World')
#> Hello World

2.5 Python has functions. 

print('Hello World')
#> Hello World

2.6 Python uses these brackets: ( ), [ ], 

print('Hello World')
#> Hello World

2.7 Python sometimes act like a prima donna when you make an error. 

print('Hello World')
#> Hello World

2.8 Python has many (many) cool packages. 

print('Hello World')
#> Hello World

2.9 You can think of the dot(‘.’) as indicating ownership. 

print('Hello World')
#> Hello World
  • Example: math.sqrt() means sqrt() belonging to math
