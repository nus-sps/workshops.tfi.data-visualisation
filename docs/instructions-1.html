5 Instructions

5.1 General flow

  1. The workshop material is composed of many Examples and Exercises.

  2. Examples:

    1. Copy and paste (use the button) the code for the examples in your Colab.

Figure 5.1: Use the Copy to Clipboard button.

  1. Run it and see if things work.

  2. Try to make sense of the code.

  3. Exercises:

    1. These are included to get you started on new features or new ways of using code.
    2. Please work through these together with your classmates and the facilitators.

5.2 If you have any difficulties

  1. Please speak (through Zoom) or use the chat (in Zoom) to raise questions and contribute to the discussion.
  2. We will not use the breakout rooms in Zoom because we like everyone to benefit from the discussions that take place.
  3. Remember that you can use Piazza to communicate with your whole class! (even after the workshop).
  4. Please use Piazza to post your questions/difficulties to the team.
    1. Please indicate the section related to your question in the question heading.
      • E.g. Basic plotting: Plot does not show?
    2. If you are having trouble with some code, please copy and paste them in your question.
    3. If necessary you can also add a screenshot.
  5. Please share code amongst yourselves too.
