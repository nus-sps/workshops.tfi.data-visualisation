1 Core Visualisation Principles

Here we would be covering the Principles of visualization. Despite the concepts being abstract, one would find it easy to improve their graphical styles in producing plots.

Core visualisation would be different to grasp has it is quite an abstract concept

Here we would try to break down the elements needed to produce the best graphical outputs.

First, one would need to try to abide by the four principles of producing a graph.

  • Minimalistic

  • Truthfulness

  • Functional

  • Beautiful

  • Inspiring

However, it would be difficult to grasp such concepts without interacting much with the data. We would be covering some concepts that have been built on top of the previous sections.
